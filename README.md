# Project

Archival of a data warehousing coursework project. The repo includes SQL, Python, and reporting assets for building and analyzing a warehouse.

## Contents
- SQL scripts for warehouse creation, master data loading, and OLAP queries (top-level and 23i2523_AbdullahAli_A/).
- Sample customer/product/transaction CSVs under Data/.
- Python utilities for hybrid join and data processing in 23i2523_AbdullahAli_A/hybrid_join/ and temp/.
- Visualization scripts for generating ETL and schema diagrams in visualization_scripts/.
- Reports and notebooks (Project-Report.tex, analysis.ipynb) plus generated figures.

## Quick start
- Python tooling: install dependencies from 23i2523_AbdullahAli_A/hybrid_join/requirements.txt (e.g., `pip install -r requirements.txt`).
- SQL: run Queries-DW.sql or the scripts inside 23i2523_AbdullahAli_A/ against your database.
